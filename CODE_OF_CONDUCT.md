# Code of Conduct

Wir verpflichten uns zu einem respektvollen, offenen und freundlichen Umgang.

## Erwartetes Verhalten
- Respektvoll kommunizieren
- Sachlich kritisieren
- Unterschiedliche Perspektiven wertschätzen
- Kein diskriminierendes, beleidigendes oder belauschendes Verhalten

## Durchsetzung
Verstöße koennen ueber Issues oder direkt an die Maintainer gemeldet werden.
