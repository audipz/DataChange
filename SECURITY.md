# Security Policy

## Supported Versions

Wir unterstuetzen nur die aktuelle Hauptversion und den neuesten Patch-Stand.

## Reporting a Vulnerability

Bitte melde Sicherheitsprobleme vertraulich ueber GitHub Security Advisories oder direkt an die Maintainer.

Bitte keine Sicherheitsprobleme oeffentlich als Issue veroeffentlichen, solange noch keine Abstimmung mit dem Maintainer erfolgt ist.
